Where do I find the mutation for the rare disease line?

Each cell line is extensively characterised. The mutation affecting any particular selected disease line, can therefore be found in the accompanying data sets. For access to these data, researchers must register and apply for access via WTSI's Electronic Data Access Mechanism.
